About This Course

Amazon Web Services (AWS) is the world’s most comprehensive and broadly adopted cloud platform, offering over 165 fully featured services from data centers globally, including compute, storage, databases, analytics, networking, mobile, developer tools, management tools, IoT, security and enterprise applications.. Millions of customers – including the fastest-growing startups, largest enterprises, and leading government agencies –trust AWS to power their infrastructure, become more agile, and lower costs This course will introduce you to Amazon Web Services (AWS) core services and infrastructure. Through demonstrations you'll learn how to use and configure AWS services and get an in-depth knowledge on EC2 instance as well as useful strategy on how to build, modify instances, host and deploy your own cloud-native applications



What will you learn

How to sign up to Amazon Web Services (AWS)


How to use AWS compute, storage, database, and security services via the AWS Console


How to make an applications durable, distributed, and highly available

AWS fundamental concepts including Regions, Availability Zones, and Virtual Private Clouds (VPCs)


How to build a simple AWS serverless system, Windows and Linux servers and create a Wordpress website


How to audit and monitor applications using services such as Amazon CloudWatch, Amazon GuardDuty and AWS Security Hub
