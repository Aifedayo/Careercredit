About the program

Our 3-month Internship program is designed to help undergraduates and fresh graduates boost their skills and build experience as Developers, or System Administrators. Training will be on relevant technologies in the field.
Developers: will be trained on Fullstack Development for Web Applications, using Python on the Django framework for back-end development, and Angular for front-end development.
System Administrators: will learn Linux Systems Administration, using CentOS 7 for Linux server administration.
Requirements: Participants must have working knowledge of HTML, Python and JavaScript.
After the first month, selected candidates will begin receiving a stipend of 20,000 Naira per week, for the remaining two months.
